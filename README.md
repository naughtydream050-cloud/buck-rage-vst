# BUCK RAGE

Industrial VST3 Plugin — Distortion/Drive + Pitch Mod/Expression

- **BUCK**: Asymmetric saturation (distortion/drive)
- **RAGE**: LFO pitch modulation (expression)
- UI: 992×496px — Asset-first rendering (BinaryData background)
- Built with JUCE 7.0.12

## Build

```bash
git submodule update --init
cmake -B build
cmake --build build --config Release
```
